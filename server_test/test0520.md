# 2026-05-20 API regression notes

pgvector embedding fuzzy search를 빼고 PostgreSQL `pg_trgm` 기반 검색으로 바꾼 뒤 확인한 내용.

기준 코드:

- name fuzzy: `NAME_TRGM`, threshold `0.60`
- address fuzzy: `ADDRESS_TRGM`, threshold `0.70`
- exact: `REGISTRATION_NUMBER`, `TAX_ID`, `SWIFT`, `PHONE`, `EMAIL`
- `SWIFT` 입력 위치: `bank_accounts[].swift`
- address-only result는 `final_score` 최대 `0.90`
- country는 단독 hit가 아니라 bonus만 줌
- `OFAC_ENT_NUM`, `ACCOUNT_NUMBER`, `IBAN`은 아직 exact 대상 아님
- response에서 `subject_type`, `source_dataset` 제거됨
